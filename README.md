# eslint-config-frbosquet-tsnext

Opinionated eslint config. Meant to be used with:

- Next app router
- Tailwind
- Typescript

## Installation

```
npm i -D eslint-config-frbosquet-tsnext
```

```
pnpm i -D eslint-config-frbosquet-tsnext
```

```
yarn add -D eslint-config-frbosquet-tsnext
```

### Usage

In your eslintrc:

```
  "extends": [
    "frbosquet-tsnext"
  ]
```
